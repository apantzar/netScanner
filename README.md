## netScanner

Welocome to netScanner!
NetScanner is a network address discovering
tool.

Runs with 
---------
```ruby
 sudo apt install python3-pip
 pip install --pre scapy
 sudo git clone https://github.com/apantzar/netScanner.git
 cd netScanner
 sudo python netScanner.py

```
